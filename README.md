## Інструкція для запуску проєкту локально

1. Спочатку потрібно клонувати репозиторій на свій локальний комп'ютер:
   git clone <url-репозиторію>

---

## Інструкція для запуску Backend

1. Перейдіть у папку backend:
   cd backend
2. Встановіть залежності:
   yarn install
3. Запустіть сервер:
   yarn start
4. Сервер доступний за адресою:
   http://localhost:3001
5. Роути:
   - GET http://localhost:3001/api/tasks → отримати список задач;
   - POST http://localhost:3001/api/tasks → додати нову задачу

---

## Інструкція для запуску Frontend

1. Перейдіть у папку frontend:
   cd frontend
2. Встановіть залежності:
   yarn install
3. Запустіть фронтенд:
   yarn dev
4. Фронтенд доступний за адресою:
   http://localhost:5173
